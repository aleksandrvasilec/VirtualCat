🐱 VirtualCat – Виртуальный питомец (котик) на 7 языках
Забавный и интерактивный виртуальный котик, который требует внимания, заботы и любви!
Кормите, играйте, лечите и укладывайте спать своего питомца, следите за его настроением и здоровьем.
Реализован на семи языках программирования с единым интерфейсом командной строки и уникальным ASCII-артом.

🚀 Возможности
Основные параметры:

🍖 Голод (0–100) – если голод высокий, котик грустный.

😊 Счастье (0–100) – влияет на игривость и здоровье.

⚡ Энергия (0–100) – нужна для игр, восстанавливается во сне.

❤️ Здоровье (0–100) – если падает ниже 30, котик может заболеть.

Действия:

feed – покормить котика (снижает голод, повышает счастье).

play – поиграть (повышает счастье, снижает энергию).

sleep – уложить спать (восстанавливает энергию, немного снижает голод).

heal – вылечить (восстанавливает здоровье, требует энергии).

Интерактивность:

Котик показывает эмоции (😺, 😿, 😴, 😼 и др.) в зависимости от состояния.

Случайные события: котик может мяукнуть, попросить еды, поцарапать мебель.

Цитаты известных людей о котах.

Сохранение – состояние питомца автоматически сохраняется в файл ~/.virtual_cat.json.

Режимы:

status – посмотреть текущее состояние котика.

rename <имя> – дать имя питомцу.

auto – автоматический режим (имитация жизни с изменением параметров каждые 10 секунд).

Цветной вывод – наглядное отображение состояния и эмоций.

📖 Использование
Синтаксис (единый для всех версий):

bash
<команда> <действие> [аргументы]
Действия
Команда	Описание
status	Показать состояние котика (все параметры, эмоцию, возраст)
feed	Покормить котика (+20 к сытости, +5 к счастью)
play	Поиграть с котиком (+15 к счастью, -10 к энергии)
sleep	Уложить котика спать (+30 к энергии, -5 к голоду)
heal	Вылечить котика (+20 к здоровью, -15 к энергии)
rename <name>	Дать котику новое имя
auto	Включить автоматический режим (параметры меняются каждые 10 сек)
Примеры
bash
# Покормить котика
python virtual_cat.py feed

# Посмотреть состояние
python virtual_cat.py status

# Дать имя Мурзик
python virtual_cat.py rename Мурзик

# Запустить автоматический режим
python virtual_cat.py auto
🛠 Установка и запуск
Python
bash
python virtual_cat.py <action> [args]
Требуется Python 3.6+.

C++
bash
g++ -std=c++17 virtual_cat.cpp -o virtual_cat -ljsoncpp
./virtual_cat <action> [args]
Требуется библиотека libjsoncpp-dev.

Go
bash
go build virtual_cat.go
./virtual_cat <action> [args]
JavaScript (Node.js)
bash
node virtual_cat.js <action> [args]
C#
bash
csc virtual_cat.cs
mono virtual_cat.exe <action> [args]   # или dotnet run
Ruby
bash
ruby virtual_cat.rb <action> [args]
Java
bash
javac virtual_cat.java
java virtual_cat <action> [args]

✨ Дополнительные фичи
ASCII-арт котика – отображается при запуске и при смене состояния.

Случайные события – мяуканье, просьба еды, игра с клубком.

Динамическое старение – котик взрослеет со временем (опция --age).

Цветные эмоции – 😺 (счастлив), 😿 (грустен), 😴 (спит), 😾 (злой), 😸 (игривый).
